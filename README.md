# android_UI_listView_XML
## 题目
使用XML定义菜单<br>
字体大小（有小，中，大这3个选项；分别对应10号字，16号字和20号字）；点击之后设置测试文本的字体<br>
普通菜单项，点击之后弹出Toast提示<br>
字体颜色（有红色和黑色这2个选项），点击之后设置测试文本的字体<br>
